Desenvolvimento Web é o termo utilizado para descrever o desenvolvimento de sites, na Internet ou numa Intranet.

O profissional que trabalha desenvolvendo websites pode ser um Web Designer (Desenvolvedor do Layout), ou Web Developer (Desenvolvedor de Sistemas).

O desenvolvimento refere-se a um processo de construção e testes do software especifico para a Web, com a finalidade de se obter um conjunto de progamas que satisfazem os funções pretendidas, quer em termos de usabilidade dos usuários ou compatilidade com outros progamas existentes.

O Desenvolvimento Web pode variar desde simples páginas estáticas a aplicações ricas, comérciod eletrônicos ou redes sociais.
